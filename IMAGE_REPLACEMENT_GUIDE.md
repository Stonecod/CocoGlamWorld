## IMPLEMENTATION STEPS

### Step 1: Update services-data.js
Replace all `imageUrl` entries in services-data.js with the corresponding URLs above.

**Example:**
```javascript
{
    category: 'Massages',
    name: 'Swedish Massage',
    imageUrl: 'https://images.unsplash.com/photo-1552821206-e4c7cb2b6d0a?w=800&q=80',
    price: 20000,
    description: 'Relaxing full body massage using long, gliding strokes.',
    duration: '60–90 minutes',
    popular: true
}
```

### Step 2: Update index.html About Section
Replace:
```html
<img src="images/aboutbackground.jpg" alt="Woman with natural beauty" />
```

## WHY THESE IMAGES?

✅ **Diverse & Inclusive**
- Show women of different skin tones and ethnicities
- Feature various body types
- Represent real, relatable beauty

✅ **Welcoming & Affordable**
- Professional but not overly luxurious/intimidating
- Warm, friendly tones
- Show expert care without premium price implications

✅ **High Quality**
- 800-1200px resolution (suitable for web)
- Good lighting and composition
- Professional photography

✅ **Relevant**
- Show actual services (massage, facials, nails, etc.)
- Depict real treatment environments
- Match Cocoglamworld's brand promise

- All URLs include `?w=XXX&q=80` parameters for optimization

- Set appropriate alt text for accessibility (SEO + WCAG compliance)
